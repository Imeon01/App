# Beispielbilder

Hier deine Beispielbilder ablegen, die in der Webapp bei "Pflanze anlegen"
als Beispielbild-Auswahl angezeigt werden (siehe `index.html`, `script.js`
`EXAMPLE_IMAGES`).

Erwartete Dateinamen (in `script.js` so eingetragen):

```
beispiel1.jpg
beispiel2.jpg
beispiel3.jpg
```

Einfach Bilddateien mit genau diesen Namen in diesen Ordner legen. Andere
Namen/Anzahl? Dann die `EXAMPLE_IMAGES`-Liste in `script.js` entsprechend
anpassen (Pfad ist relativ zu `index.html`, z. B.
`"example-images/mein-foto.jpg"`).

Empfehlung für gute PlantNet-Trefferquote: pro Pflanze möglichst
unterschiedliche Pflanzenteile (z. B. Blatt + Blüte), keine reinen
Duplikate — PlantNet nutzt bis zu 3 Bilder pro Bestimmung.
